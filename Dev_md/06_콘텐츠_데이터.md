### 데이터 구조
```javascript
{
  id: 'basics',
  title: '프레젠테이션 기초',
  icon: '📋',
  description: '프레젠테이션의 기본 개념과 기획 방법을 배웁니다',
  topics: [
    {
      id: 'what-is-presentation',
      title: '프레젠테이션이란?',
      sections: [
        {
          title: '프레젠테이션의 정의',
          content: '마크다운 텍스트 (표, 볼드, 코드, 인용 등)'
        },
        { title: '프레젠테이션의 유형', content: '...' }
      ]
    }
  ]
}
```

### 헬퍼 함수
- `getLearningCategory(categoryId)` — ID로 카테고리 검색
- `getLearningTopic(categoryId, topicId)` — 카테고리+토픽 ID로 토픽 검색
