# 콘텐츠 데이터

## 커리큘럼 데이터 (src/data/curriculum.js)

### 5개 과정 구조

| # | slug | 제목 | 기간 | 난이도 |
|---|------|------|------|--------|
| 1 | planning | 프레젠테이션 기획과 구성 | 4주 | 입문 |
| 2 | ppt-design | PPT 디자인 원칙 | 4주 | 초급 |
| 3 | speech | 스피치와 전달력 향상 | 4주 | 초급 |
| 4 | data-visualization | 데이터 시각화 기법 | 3주 | 중급 |
| 5 | practice | 실전 발표 연습 | 3주 | 중급 |

### 각 과정 데이터 필드
- `id`: 순번
- `slug`: URL 경로
- `title`: 전체 제목
- `shortTitle`: 짧은 제목 (네비게이션용)
- `icon`: 이모지 아이콘
- `description`: 과정 설명
- `topics`: 주요 학습 키워드 (5개)
- `modules`: 학습 모듈 배열 (각 모듈에 title, topics)
- `duration`: 학습 기간
- `level`: 난이도
- `features`: 학습 특징 (3개)

### 총 모듈/토픽 수
- 총 20개 모듈
- 총 60+ 개 세부 토픽

## 도구 데이터 (src/data/tools.js)

### 6개 도구

| # | id | 도구명 | 개발사 |
|---|-----|--------|--------|
| 1 | powerpoint | Microsoft PowerPoint | Microsoft |
| 2 | google-slides | Google Slides | Google |
| 3 | canva | Canva | Canva Pty Ltd |
| 4 | miricanvas | 미리캔버스 | 미리디 (Miridi) |
| 5 | figma | Figma | Figma, Inc. |
| 6 | genially | Genially (젠리) | Genially Web S.L. |

### 각 도구 데이터 필드
- `id`: URL 경로용 ID
- `name`: 전체 이름
- `shortName`: 짧은 이름
- `company`: 개발사
- `icon`: 이모지 아이콘
- `iconClass`: CSS 아이콘 클래스
- `color`: 브랜드 컬러
- `gradient`: 그라디언트 배경
- `description`: 짧은 설명
- `fullDescription`: 상세 설명
- `tags`: 분류 태그 (4개)
- `pricing`: 요금제 배열 (plan, price, period, features, featured)
- `features`: 주요 기능 (8개)
- `pros`: 장점 (5개)
- `cons`: 단점 (4개)
- `useCases`: 추천 활용 분야 (4개)
- `officialUrl`: 공식 사이트 URL
- `tips`: 활용 팁 (5개)

### 도구 비교표 (toolComparison)
11개 항목 기준 6개 도구 비교:
- 가격, 오프라인 사용, 공동 작업, 디자인 난이도, 템플릿 수
- 애니메이션, 한글 지원, 인터랙티브, 비디오 내보내기, AI 기능, 추천 대상
