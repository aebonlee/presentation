# 라우팅 구조

## 라우터 설정
- **라이브러리**: React Router DOM v7
- **방식**: 클라이언트 사이드 SPA 라우팅
- **코드 분할**: React.lazy() + Suspense (모든 페이지)
- **로딩**: 스피너 컴포넌트

## 라우트 목록

### 공개 라우트 (PublicLayout)

| 경로 | 컴포넌트 | 설명 |
|------|---------|------|
| `/` | Home | 홈페이지 (히어로, 커리큘럼, 도구, 특징, 대상, CTA) |
| `/about` | About | 프로그램 소개 |
| `/login` | Login | Google/카카오 OAuth 로그인 |
| `/curriculum/:slug` | CourseDetail | 커리큘럼 상세 (동적) |
| `/tools` | ToolsOverview | 도구 비교 페이지 |
| `/tools/:toolId` | ToolDetail | 도구 상세 (동적) |
| `*` | NotFound | 404 페이지 |

### 커리큘럼 slug 값
| slug | 페이지 |
|------|--------|
| `planning` | 프레젠테이션 기획과 구성 |
| `ppt-design` | PPT 디자인 원칙 |
| `speech` | 스피치와 전달력 향상 |
| `data-visualization` | 데이터 시각화 기법 |
| `practice` | 실전 발표 연습 |

### 도구 toolId 값
| toolId | 페이지 |
|--------|--------|
| `powerpoint` | Microsoft PowerPoint |
| `google-slides` | Google Slides |
| `canva` | Canva |
| `miricanvas` | 미리캔버스 |
| `figma` | Figma |
| `genially` | Genially (젠리) |

## GitHub Pages SPA 지원
- `public/404.html`: 경로를 쿼리스트링으로 변환하여 리다이렉트
- `index.html` 내 스크립트: sessionStorage.redirect로 경로 복원
- `public/CNAME`: presentation.dreamitbiz.com

## 레이아웃 구조
```
PublicLayout
├── Navbar (고정 네비게이션)
├── <Suspense fallback={Loading}>
│   └── <Outlet /> (라우트 페이지)
└── Footer
```
