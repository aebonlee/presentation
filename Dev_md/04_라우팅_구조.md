# 라우팅 구조

## 라우터 설정
- **라이브러리**: React Router DOM v7
- **방식**: 클라이언트 사이드 SPA 라우팅
- **코드 분할**: React.lazy() + Suspense (모든 페이지)
- **로딩**: 스피너 컴포넌트

## 전체 라우트 맵

```
<Routes>
  <Route element={<PublicLayout />}>
    ├── /                              → Home (홈)
    ├── /about                         → About (소개)
    ├── /login                         → Login (로그인)
    │
    │  [학습 페이지]
    ├── /learn                         → LearningPage (학습 메인)
    ├── /learn/:categoryId             → LearningPage (카테고리별)
    ├── /learn/:categoryId/:topicId    → LearningPage (토픽별)
    │
    │  [보조 학습]
    ├── /enrollment                    → Enrollment (과정 신청)
    ├── /glossary                      → Glossary (용어사전)
    ├── /practice                      → Practice (퀴즈)
    │
    │  [커리큘럼/도구]
    ├── /curriculum/:slug              → CourseDetail (과정 상세)
    ├── /tools                         → ToolsOverview (도구 비교)
    ├── /tools/:toolId                 → ToolDetail (도구 상세)
    │
    └── /*                             → NotFound (404)
  </Route>
</Routes>
```

총 **13개 라우트** (PublicLayout 래퍼 포함)

## 네비게이션 구조

```
홈  |  학습 ▾  |  도구 ▾  |  용어사전  |  퀴즈  |  과정 신청  |  [로그인]

학습 드롭다운:
  ├── 프레젠테이션 기초  → /learn/basics
  ├── 슬라이드 디자인    → /learn/slide-design
  ├── 스피치와 전달력    → /learn/speech
  ├── 데이터 시각화      → /learn/data-viz
  ├── 스토리텔링         → /learn/storytelling
  └── 영어 프레젠테이션  → /learn/english-presentation

도구 드롭다운:
  ├── 도구 비교      → /tools
  ├── PowerPoint     → /tools/powerpoint
  ├── Google Slides  → /tools/google-slides
  ├── Canva          → /tools/canva
  ├── 미리캔버스     → /tools/miricanvas
  ├── Figma          → /tools/figma
  └── Genially       → /tools/genially
```

## GitHub Pages SPA 지원
- `public/404.html`: 경로를 쿼리스트링으로 변환하여 리다이렉트
- `index.html` 내 스크립트: sessionStorage.redirect로 경로 복원
- `public/CNAME`: presentation.dreamitbiz.com

├── <Suspense fallback={Loading}>

└── Footer (4컬럼)
