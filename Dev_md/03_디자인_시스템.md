# 디자인 시스템

## 디자인 접근 방식
- **순수 CSS** + **CSS Custom Properties** (회사 메인 사이트 패턴 동일)
- Tailwind, styled-components 등 미사용
- data-theme, data-color 속성으로 동적 테마 전환
- Glassmorphism 디자인 패턴 적용
- OpenClaw 스타일 사이드바 학습 페이지 디자인 참조

## 기본 컬러 (다크 블루)

### Primary Colors
```css
--primary: #1E3A5F         /* 다크 블루 */
--primary-dark: #0A1628    /* 깊은 다크 블루 */
--primary-light: #2E5A8E   /* 라이트 블루 */
--primary-lighter: #4A8FE7 /* 더 밝은 블루 */
```

### Accent Colors
```css
--accent: #2563EB          /* 메인 액센트 블루 */
--accent-dark: #1D4ED8     /* 진한 액센트 */
--accent-light: #60A5FA    /* 밝은 액센트 */
```

## 5가지 컬러 테마

| # | 테마명 | Primary | Accent | data-color |
|---|--------|---------|--------|------------|
| 1 | **Blue** (기본) | #1E3A5F | #2563EB | blue |
| 2 | **Teal** | #134E4A | #0D9488 | teal |
| 3 | **Purple** | #3B1E6F | #7C3AED | purple |
| 4 | **Emerald** | #14532D | #059669 | emerald |
| 5 | **Gold** | #4A3400 | #D97706 | gold |

## 다크/라이트 모드

| 모드 | 설명 | data-theme |
|------|------|------------|
| Light | 흰색 배경, 어두운 텍스트 | light |
| Dark | 다크 블루 배경, 밝은 텍스트 | dark |
| Auto | 시간대 기반 자동 전환 (06:00~18:00 라이트) | auto → light/dark |

저장: `localStorage` prs_themeMode, prs_colorTheme

## CSS 파일 구성 (12개)

| 파일 | 용도 |
|------|------|
| base.css | 디자인 토큰, 리셋, 유틸리티, 버튼, 카드, 배지, 그리드 |
| dark-mode.css | 다크 테마 오버라이드 |
| navbar.css | 고정 네비바, 드롭다운, 컬러피커, 모바일 메뉴 |
| hero.css | 풀뷰포트 히어로, 파티클, 스크롤 인디케이터 |
| footer.css | 4컬럼 푸터, 패밀리 사이트 |
| curriculum.css | 커리큘럼 카드, 과정 상세 (2컬럼 + sticky 사이드바) |
| tools.css | 도구 카드, 가격표, 비교표, 도구 상세 |
| auth.css | 로그인 카드, 소셜 버튼, 토스트 알림 |
| pages.css | 기능, CTA, 소개, 대상, 404, 로딩 |
| animations.css | AOS 효과 (fade-up/down/left/right, zoom-in) |
| **learning.css** | 학습 사이드바, 마크다운 렌더링, 수강신청 카드, 용어사전, 퀴즈 |
| responsive.css | 반응형 브레이크포인트 (1280/1024/768/480px) |

## 반응형 브레이크포인트
| 브레이크포인트 | 대상 |
|---------------|------|
| 1280px | 대형 데스크톱 |
| 1024px | 데스크톱/태블릿 (모바일 메뉴 전환, 학습 사이드바 토글) |
| 768px | 태블릿/모바일 |
| 480px | 소형 모바일 |

## 애니메이션 (AOS)
- Intersection Observer 기반
- 효과: fade-up, fade-down, fade-left, fade-right, zoom-in
- 딜레이: 100ms ~ 500ms
- 이징: cubic-bezier(0.4, 0, 0.2, 1)
- 이동 거리: 24px
