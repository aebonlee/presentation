# 03. 디자인 시스템

## 디자인 접근 방식
- **순수 CSS** + **CSS Custom Properties** (회사 메인 사이트 패턴 동일)
- Tailwind, styled-components 등 미사용
- `data-theme`, `data-color` 속성으로 동적 테마 전환
- Glassmorphism 디자인 패턴 적용
- OpenClaw 스타일 사이드바 학습 페이지 디자인 참조

## 컨테이너 너비 (3단계)

| 변수 | 너비 | 용도 |
|------|------|------|
| `--container-max` | 1200px | 메인 레이아웃, 네비바, 학습 페이지 |
| `--container-content` | 860px | 콘텐츠 영역 |
| `--container-narrow` | 640px | 좁은 텍스트 블록, 히어로 설명 |

```css
.container         { max-width: var(--container-max); }
.container-content { max-width: var(--container-content); }
.container-narrow  { max-width: var(--container-narrow); }
```

## 5가지 컬러 테마

`data-color` 속성으로 전환. 각 테마는 `--primary`, `--accent` 계열 변수를 오버라이드.

| # | 테마명 | Accent | Primary Dark | data-color |
|---|--------|--------|-------------|------------|
| 1 | **Blue** (기본) | `#2563EB` | `#0A1628` | blue |
| 2 | **Teal** | `#0D9488` | `#042F2E` | teal |
| 3 | **Purple** | `#7C3AED` | `#1E0A3E` | purple |
| 4 | **Emerald** | `#059669` | `#052E16` | emerald |
| 5 | **Gold** | `#D97706` | `#2A1D00` | gold |

## 다크/라이트 모드

| 모드 | 설명 | data-theme |
|------|------|------------|
| Light | 흰색 배경, 어두운 텍스트 | light |
| Dark | 다크 블루 배경, 밝은 텍스트 (`dark-mode.css` 오버라이드) | dark |
| Auto | 시간대 기반 자동 전환 (06:00~18:00 라이트) | auto → light/dark |

저장: `localStorage` — `prs_themeMode`, `prs_colorTheme`

## 컴포넌트 토큰

### Border Radius
```css
--radius-sm: 8px;  --radius-md: 12px;  --radius-lg: 16px;
--radius-xl: 20px;  --radius-2xl: 24px;  --radius-full: 9999px;
```

### Shadow
```css
--shadow-sm:   0 1px 3px rgba(0,0,0,0.06);
--shadow-md:   0 4px 6px rgba(0,0,0,0.06), 0 2px 4px rgba(0,0,0,0.04);
--shadow-lg:   0 10px 25px rgba(0,0,0,0.08);
--shadow-xl:   0 20px 40px rgba(0,0,0,0.1);
--shadow-glow: 0 0 30px var(--accent-glow);
```

### Glass Effect
```css
--glass-bg: rgba(255,255,255,0.85);
--glass-blur: 12px;
--glass-border: rgba(255,255,255,0.2);
```

### Transition
```css
--transition-fast:  0.15s ease;
--transition-base:  0.3s ease;
--transition-slow:  0.5s ease;
--transition-cubic: 0.3s cubic-bezier(0.4, 0, 0.2, 1);
```

## 애니메이션 (AOS)
- Intersection Observer 기반 (`useAOS` 커스텀 훅)
- 효과: fade-up, fade-down, fade-left, fade-right, zoom-in
- 딜레이: 100ms ~ 500ms
- 이징: cubic-bezier(0.4, 0, 0.2, 1)
- 이동 거리: 24px

## CSS 파일 구성 (14개)

| 파일 | 용도 |
|------|------|
| `base.css` | 디자인 토큰, 리셋, 유틸리티, 버튼, 카드, 배지, 그리드 |

| `footer.css` | 4컬럼 푸터, 패밀리 사이트 셀렉트 |
| `curriculum.css` | 커리큘럼 카드, 과정 상세 (2컬럼 + sticky) |
| `tools.css` | 도구 카드, 가격표, 비교표, 유즈케이스, 팁 |
| `auth.css` | 로그인 카드, 소셜 OAuth 버튼, 토스트 알림 |

| `animations.css` | AOS 효과 (fade, zoom) |
| `learning.css` | 학습 사이드바, 마크다운, 수강신청, 용어사전, 퀴즈, 비주얼 다이어그램 |
| `tool-guide.css` | 제작도구 종합 가이드 (사이드바, 탭, 마크다운, 도구 소개 통합) |
| `community.css` | 커뮤니티 게시판 (목록, 상세, 폼, 댓글, 좋아요, 페이지네이션) |
