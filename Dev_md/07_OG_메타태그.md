# OG 메타태그 및 SEO

## Open Graph 설정

### 기본 메타태그 (index.html)
```html
<meta property="og:title" content="DreamIT 프레젠테이션 | 효과적인 발표 기법과 시각 자료 제작" />
<meta property="og:description" content="PPT, 스피치, 시각디자인, 스토리텔링 학습" />
<meta property="og:type" content="website" />
<meta property="og:url" content="https://presentation.dreamitbiz.com" />
<meta property="og:image" content="https://presentation.dreamitbiz.com/og-image.png" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
<meta property="og:locale" content="ko_KR" />
<meta property="og:site_name" content="DreamIT 프레젠테이션" />
```

### Twitter Card
```html
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="..." />
<meta name="twitter:description" content="..." />
<meta name="twitter:image" content="..." />
```

## SEOHead 컴포넌트 (src/components/SEOHead.jsx)
React 19+ native head hoisting 사용.

### Props
| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| title | string | - | 페이지 제목 (사이트명 자동 추가) |
| description | string | 기본 설명 | 페이지 설명 |
| path | string | '' | URL 경로 |
| ogImage | string | og-image.png | OG 이미지 URL |
| noindex | boolean | false | 검색 엔진 색인 제외 |

## OG 이미지 생성

### 스크립트: scripts/generate-og-image.js
- **기술**: sharp (SVG → PNG 변환)
- **크기**: 1200 x 630px
- **디자인**: 다크 블루 그라디언트 배경, 브랜드 로고, 태그 표시
- **실행**: `npm run generate-og`

### 카카오 공유 디버깅
- https://developers.kakao.com/tool/debugger/sharing 에서 확인 가능
