### 사용 기술 결정 이유
| 결정 | 이유 |
|------|------|
| React 19 + Vite 7 | 회사 메인 사이트와 동일 스택 |
| 순수 CSS + CSS Variables | 메인 사이트 패턴 일관성 |
| Supabase | 메인 사이트와 동일 인프라 |
| Google + 카카오 OAuth | 한국 사용자 대상 |
| sharp (OG 이미지) | 서버 없이 빌드 시 이미지 생성 |
| AOS (Intersection Observer) | 메인 사이트와 동일 애니메이션 방식 |
| Lazy Loading | 성능 최적화 (코드 스플리팅) |
