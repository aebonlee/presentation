## Phase 6: 제작도구 종합 가이드 시스템 (신규)

### 개발 배경
기존 도구 소개 페이지(`ToolDetail`)가 기능 목록, 가격, 장단점, 5개 팁 수준에 머물러 있어, 각 도구를 실제로 활용하기 위한 **종합 사용 가이드**가 필요.

### 아키텍처 결정
| 결정 | 이유 |
|------|------|
| 별도 `tool-guides/` 디렉토리 | 도구당 500~1300줄 → `tools.js`에 추가 시 5000줄+ 비대화 방지 |
| 새 `ToolGuide.jsx` 페이지 | 기존 ToolDetail은 소개 유지, 별도 가이드 페이지로 분리 |
| `renderMarkdown.js` 유틸리티 추출 | LearningPage 내부 함수 → 공유 유틸리티로 추출하여 중복 제거 |
| LearningPage 패턴 재사용 | 사이드바+섹션탭+마크다운 렌더링 구조 동일 |

### 신규 생성 파일 (10개)
```
src/utils/renderMarkdown.js          # 마크다운 렌더러 (LearningPage에서 추출)
src/styles/tool-guide.css            # 가이드 전용 CSS (~350줄)
src/pages/tools/ToolGuide.jsx        # 가이드 페이지 컴포넌트
src/data/tool-guides/index.js        # 어그리게이터 + getGuideByToolId()
src/data/tool-guides/powerpoint.js   # PowerPoint 가이드
src/data/tool-guides/google-slides.js # Google Slides 가이드
src/data/tool-guides/canva.js        # Canva 가이드
src/data/tool-guides/miricanvas.js   # 미리캔버스 가이드
src/data/tool-guides/figma.js        # Figma 가이드
src/data/tool-guides/genially.js     # Genially 가이드
```
