# 빌드 및 배포

## 스크립트

```bash
npm run dev          # 개발 서버 (localhost:5174)
npm run build        # 프로덕션 빌드 → dist/
npm run preview      # 빌드 결과 미리보기
npm run lint         # ESLint 검사
npm run generate-og  # OG 이미지 재생성
```

## 빌드 결과
- 출력 디렉토리: `dist/`
- 에셋 디렉토리: `dist/assets/`
- CSS: ~39KB (gzip ~7.4KB)
- JS (메인): ~436KB (gzip ~129KB, React 포함)
- 페이지 JS: 각 0.3~15.8KB

## 배포: GitHub Pages (gh-pages)

### 방법
1. `npm run build`로 dist/ 생성
2. gh-pages 패키지를 사용하여 dist/ 폴더를 gh-pages 브랜치에 배포
3. GitHub 리포지토리 Settings > Pages에서 gh-pages 브랜치 선택

### 커스텀 도메인
- `public/CNAME`: `presentation.dreamitbiz.com`
- DNS: CNAME 레코드 → `aebonlee.github.io`

### SPA 라우팅 지원
- `public/404.html`: 경로를 쿼리스트링으로 인코딩하여 index.html로 리다이렉트
- `index.html`: sessionStorage.redirect로 원래 경로 복원

## Vite 설정 (vite.config.js)
```javascript
{
  plugins: [react()],
  base: '/',
  build: {
    outDir: 'dist',
    assetsDir: 'assets',
    emptyOutDir: true
  },
  server: {
    host: true,
    port: 5174
  }
}
```
