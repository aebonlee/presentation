# 컴포넌트 상세

## 레이아웃 컴포넌트

### Navbar (src/components/layout/Navbar.jsx)
- **위치**: 고정 (position: fixed)
- **스타일**: 투명 → 스크롤 시 글래스모피즘 전환
- **기능**:
  - 스크롤 감지 (30px 이후 배경 전환)
  - 드롭다운 메뉴: **학습** (6개 카테고리), **도구** (도구 비교 + 6개 도구)
  - 데스크톱: hover 드롭다운 / 모바일: 클릭 토글 (1024px 전환)
  - 독립 링크: 용어사전, 퀴즈, 과정 신청
  - 테마 토글 (☀️ → 🌙 → 🔄)
  - 컬러 피커 (5색 선택)
  - 사용자 메뉴 (로그인 시 아바타, 로그아웃)
  - 로그인 버튼 (미인증 시)
  - 햄버거 메뉴 (모바일)

### Footer (src/components/layout/Footer.jsx)
- **구조**: 4컬럼 그리드
  1. 브랜드 + 설명 + 패밀리 사이트 셀렉트박스 (5개)
  2. 학습 링크 (6개 카테고리 + 용어사전 + 퀴즈)
  3. 도구 학습 링크 (6개 도구)
  4. 연락처 (이메일, 전화, 카카오톡, 영업시간)
- **하단**: 저작권 + 사업자 정보

### PublicLayout (src/layouts/PublicLayout.jsx)
- Navbar + Suspense(Outlet) + Footer
- 로딩 시 스피너 표시

## 핵심 페이지: 학습

### LearningPage (src/pages/learning/LearningPage.jsx)
OpenClaw 스타일 사이드바 학습 페이지. 사이트의 **핵심 컴포넌트**.

- **URL**: `/learn`, `/learn/:categoryId`, `/learn/:categoryId/:topicId`
- **사이드바** (280px, sticky):
  - "학습 목차" 헤더
  - 6개 카테고리 목록 (아이콘 + 제목), 활성 카테고리 하이라이트
  - 활성 카테고리의 토픽 목록 (들여쓰기)
  - 모바일: 토글 버튼으로 열기/닫기
- **콘텐츠 영역**:
  - 브레드크럼 (홈 › 카테고리 › 토픽)
  - 헤더 (아이콘, 제목, 설명)
  - 섹션 탭 (2개 이상일 때 표시)
  - **마크다운 렌더러**: H3, 볼드, 인라인 코드, 코드블록, 인용, 표, 순서/비순서 리스트, 취소선, 단락
  - 하단 네비게이션 (이전/다음 섹션, 다음 토픽/카테고리)
- **상태**: activeSection, sidebarOpen
- **효과**: categoryId/topicId 변경 시 섹션 초기화 + 스크롤 맨 위

## 보조 학습 페이지

### Enrollment (src/pages/Enrollment.jsx)
과정 신청 페이지. 기존 5개 커리큘럼 데이터를 수강 신청 UI로 활용.

- **히어로 섹션**: "과정 신청" 타이틀
- **과정 카드** (5개): enrollment-card CSS 클래스 사용
  - 아이콘 + 제목 + 난이도/기간 메타
  - 설명
  - 접이식 모듈 목록 (▶ 커리큘럼 보기, 클릭 시 모듈/토픽 펼치기)
  - 기능 배지 (feature tags)
  - 액션 버튼: "상세보기" (Link → /curriculum/:slug), "신청하기" (alert)
- **수강 신청 안내**: 3단계 프로세스 카드 (과정 선택 → 신청서 제출 → 수강 시작)
- **문의 정보**: 이메일, 전화

### Glossary (src/pages/Glossary.jsx)
용어사전 페이지. 36개 프레젠테이션 전문 용어.

- **검색**: glossary-search-input으로 용어/영문/정의 필터링 (대소문자 무시)
- **카테고리 필터**: 7개 버튼 (전체, 기획, 디자인, 스피치, 데이터, 도구, 영어표현)
- **결과 카운트**: "총 N개 용어" (accent 색상 강조)
- **용어 카드**: glossary-card (한글 용어, 영문명, 정의)
- **빈 상태**: 검색 결과 없음 메시지

### Practice (src/pages/Practice.jsx)
퀴즈 페이지. 20문제 프레젠테이션 지식 테스트.

- **진행 표시**: "X / 20" + 현재 점수 + 프로그레스 바 (accent 색상)
- **문제 카드**: quiz-card + quiz-question + quiz-options
- **옵션 선택**: 클릭 → 정답(초록)/오답(빨강) 스타일 + 해설 표시
- **다음 문제 / 결과 보기** 버튼
- **완료 화면**: 점수(X/20), 정답률(%), 등급 메시지 (5단계), "다시 풀기" / "커리큘럼 학습하기" 버튼
- **상태**: currentQuestion, selectedAnswer, showResult, score, isFinished
