# 08. 컴포넌트 상세

## 레이아웃 컴포넌트

### Navbar (src/components/layout/Navbar.jsx)
- **위치**: 고정 (position: fixed, z-index: 1000)
- **높이**: `var(--nav-height)` = 72px
- **브랜드**: `[P] Presentation` (로고 아이콘 + 텍스트)
- **스타일 전환**:
  - `navbar-transparent` → `navbar-scrolled` (스크롤 30px 이후)
  - `/learn/*`, `/login`, `/tools/:toolId` 경로: 항상 `navbar-scrolled` (noHeroPage 감지)
- **메뉴 구성**:
  - `A ~ Z 학습 ▾` — 드롭다운 (6개 학습 카테고리)
  - `학습 강의안` — 독립 링크 (`/lectures`)
  - `제작도구 ▾` — 드롭다운 (도구 비교 + 6개 도구)
  - `참고 자료 ▾` — 드롭다운 (용어사전, 퀴즈)
  - `과정 신청`, `제작 의뢰`, `커뮤니티` — 독립 링크
- **우측 컨트롤**:
  - 테마 토글: ☀️ (Light) → 🌙 (Dark) → 🔄 (Auto)
  - 컬러 피커: 🎨 → 드롭다운 (5색 도트)
  - 로그인 버튼 (미인증) / 사용자 아바타 + 드롭다운 (인증)
  - 햄버거 메뉴 (모바일 1024px 이하)
- **드롭다운 동작**:
  - 데스크톱: hover 열기/닫기
  - 모바일: 클릭 토글
  - 중앙 정렬: `left: 50%; transform: translateX(-50%)`

### Footer (src/components/layout/Footer.jsx)

- **하단**: 저작권 (`© 2025 드림아이티비즈(DreamIT Biz)`) + `Designed by Ph.D Aebon Lee | 대표이사: 이애본 | 사업자등록번호: 601-45-20154`

- **레이아웃**: `max-width: var(--container-max); margin: 0 auto;` (중앙 정렬)
- **사이드바** (280px, sticky):
  - "학습 목차" 헤더
  - 6개 카테고리 목록 (아이콘 + 제목), 활성 카테고리 하이라이트

  - 브레드크럼 (홈 > 카테고리 > 토픽)
  - 헤더 (아이콘, 제목, 설명)

### ToolGuide (src/pages/tools/ToolGuide.jsx) — 통합 도구 페이지
- 동적: `/tools/:toolId` (도구 소개 + 종합 가이드 통합)
- LearningPage와 유사한 사이드바+콘텐츠 레이아웃
- **INTRO_SECTION**: 도구 소개를 가이드 첫 번째 섹션(인덱스 0)으로 삽입
  - 5개 서브섹션: 도구 개요, 요금제, 주요 기능·장단점, 추천 활용 분야, 활용 팁
  - `IntroContent` 컴포넌트: 기존 ToolDetail의 리치 JSX 렌더링 (마크다운 아님)
- **allSections**: `[INTRO_SECTION, ...guide.sections]` (총 6섹션)
- **사이드바** (280px, sticky):
  - 도구 목록 복귀 링크 (`← 도구 목록으로 돌아가기`)
  - 도구 아이콘 + 이름
  - 6개 섹션 목록: 섹션명 버튼 + ▸ 화살표 토글 버튼 분리
  - **아코디언 드롭다운**: `expandedSection` (단일 값) 상태로 1개만 열림, 나머지 자동 닫힘
  - 화살표 클릭 → 토글, 섹션명 클릭 → 해당 섹션으로 이동 + 자동 펼침
  - 모바일: ☰ 토글 버튼
- **콘텐츠 영역**:
  - 브레드크럼 (홈 > 제작도구 > 도구명 가이드)
  - 헤더 (도구 그래디언트 아이콘 + 제목)
  - 서브섹션 탭 바 (2개 이상일 때 표시)
  - **상단 네비게이션** (`.tool-guide-nav-top`): 이전/다음 서브섹션→섹션 이동 버튼
  - 콘텐츠 렌더링 분기: `activeSection === 0` → IntroContent (JSX), 나머지 → `renderMarkdown`
  - **하단 도구 네비게이션** (`.tool-guide-tool-nav`): 이전/다음 도구 + 전체 비교 링크
- **상태**: `activeSection`, `activeSubsection`, `sidebarOpen`, `expandedSection` (단일 값)
- **데이터**: `src/data/tool-guides/` + `src/data/tools.js`
- **CSS**: `src/styles/tool-guide.css`
