# 컴포넌트 상세

## 레이아웃 컴포넌트

### Navbar (src/components/layout/Navbar.jsx)
- **위치**: 고정 (position: fixed)
- **스타일**: 투명 → 스크롤 시 글래스모피즘 전환
- **기능**:
  - 스크롤 감지 (30px 이후 배경 전환)
  - 드롭다운 메뉴 (커리큘럼 5개, 도구 6개)
  - 데스크톱: hover 드롭다운
  - 모바일: 클릭 토글 (1024px 브레이크포인트)
  - 테마 토글 (☀️ → 🌙 → 🔄)
  - 컬러 피커 (5색 선택)
  - 사용자 메뉴 (로그인 시 아바타, 로그아웃 버튼)
  - 로그인 버튼 (미인증 시)
  - 햄버거 메뉴 (모바일)

### Footer (src/components/layout/Footer.jsx)
- **구조**: 4컬럼 그리드
  1. 브랜드 + 설명 + 패밀리 사이트 셀렉트박스
  2. 커리큘럼 링크 5개
  3. 도구 학습 링크 6개
  4. 연락처 (이메일, 전화, 카카오톡, 영업시간)
- **하단**: 저작권 + 사업자 정보

### PublicLayout (src/layouts/PublicLayout.jsx)
- Navbar + Suspense(Outlet) + Footer
- 로딩 시 스피너 표시

## 페이지 컴포넌트

### Home (src/pages/Home.jsx)
5개 섹션으로 구성:
1. **Hero**: 파티클 애니메이션, 타이틀, CTA 버튼, 통계 (5 모듈, 6 도구, 20+ 프로젝트, 100% 실전)
2. **커리큘럼**: 5개 과정 카드 그리드
3. **도구**: 6개 도구 카드 그리드
4. **특징**: 템플릿 제공, 영상 피드백, 실전 시뮬레이션
5. **대상**: 대학생, 직장인, 교육자, 창업자
6. **CTA**: 학습 시작 유도

### CourseDetail (src/pages/courses/CourseDetail.jsx)
- 동적 라우트: `/curriculum/:slug`
- **Hero**: 과정 정보, 기간, 난이도, 모듈 수
- **Content**: 2컬럼 (본문 + 사이드바)
  - 본문: 학습 개요, 모듈별 토픽 목록, 키워드 태그
  - 사이드바 (sticky): 과정 정보, 학습 특징, 전체 커리큘럼 네비
- **네비게이션**: 이전/다음 과정 이동

### ToolsOverview (src/pages/tools/ToolsOverview.jsx)
- **Hero**: 제목, 설명
- **도구 카드**: 6개 도구 그리드
- **비교표**: 11개 항목 × 6개 도구 테이블
- **추천**: 상황별 6개 추천 카드 (비즈니스, 학생, 한국어, 디자인, 인터랙티브, 스타트업)

### ToolDetail (src/pages/tools/ToolDetail.jsx)
- 동적 라우트: `/tools/:toolId`
- **Hero**: 도구 브랜드 그라디언트 배경, 태그, 공식 사이트 링크
- **요금제**: 가격 카드 (추천 플랜 하이라이트)
- **기능/장점/단점**: 3컬럼 정보 카드
- **활용 분야**: 추천 사용처
- **팁**: 5개 실전 팁 목록
- **네비게이션**: 이전/다음 도구 이동

### Login (src/pages/Login.jsx)
- 중앙 정렬 카드 레이아웃
- Google 로그인 버튼 (SVG 아이콘)
- 카카오 로그인 버튼 (SVG 아이콘)
- 로그인 완료 시 홈으로 자동 리다이렉트

### About (src/pages/About.jsx)
- **Hero**: 프로그램 소개
- **특징**: 4개 카드 (커리큘럼, 도구, 영상 피드백, 시뮬레이션)
- **로드맵**: 18주 타임라인 (5단계)
- **CTA**: 학습 시작 유도

### NotFound (src/pages/NotFound.jsx)
- 404 그라디언트 텍스트
- 홈 링크 버튼
