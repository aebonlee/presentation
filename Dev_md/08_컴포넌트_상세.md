# 08. 컴포넌트 상세

## 레이아웃 컴포넌트

### Navbar (src/components/layout/Navbar.jsx)
- **위치**: 고정 (position: fixed, z-index: 1000)
- **높이**: `var(--nav-height)` = 72px
- **브랜드**: `[P] Presentation` (로고 아이콘 + 텍스트)
- **스타일 전환**:
  - `navbar-transparent` → `navbar-scrolled` (스크롤 30px 이후)
  - `/learn/*`, `/login` 경로: 항상 `navbar-scrolled` (noHeroPage 감지)
- **메뉴 구성**:
  - `A ~ Z 학습 ▾` — 드롭다운 (6개 학습 카테고리)
  - `제작도구 ▾` — 드롭다운 (도구 비교 + 6개 도구)
  - `용어사전`, `퀴즈`, `과정 신청`, `제작 의뢰`, `커뮤니티` — 독립 링크
- **우측 컨트롤**:
  - 테마 토글: ☀️ (Light) → 🌙 (Dark) → 🔄 (Auto)
  - 컬러 피커: 🎨 → 드롭다운 (5색 도트)
  - 로그인 버튼 (미인증) / 사용자 아바타 + 드롭다운 (인증)
  - 햄버거 메뉴 (모바일 1024px 이하)
- **드롭다운 동작**:
  - 데스크톱: hover 열기/닫기
  - 모바일: 클릭 토글
  - 중앙 정렬: `left: 50%; transform: translateX(-50%)`

### Footer (src/components/layout/Footer.jsx)
- **구조**: 4컬럼 그리드
  1. 브랜드 (`DreamIT 프레젠테이션`) + 설명 + 패밀리 사이트 셀렉트 (5개 사이트)
  2. 학습 링크 (6개 카테고리 + 용어사전 + 퀴즈)
  3. 도구 학습 링크 (6개 도구)
  4. 연락처 (✉️ 이메일, 📞 전화, 💬 카카오톡, 🕐 영업시간)
- **하단**: 저작권 (`© 2025 드림아이티비즈`) + 사업자 정보

### PublicLayout (src/layouts/PublicLayout.jsx)
- `Navbar` + `<Suspense fallback={<Loading />}>` + `<Outlet />` + `Footer`

### SEOHead (src/components/SEOHead.jsx)
- React 19+ native head hoisting
- Props: `title`, `description`, `path`, `ogImage`, `noindex`

## 핵심 페이지: 학습

### LearningPage (src/pages/learning/LearningPage.jsx)
OpenClaw 스타일 사이드바 학습 페이지. 사이트의 **핵심 컴포넌트**.

- **URL**: `/learn`, `/learn/:categoryId`, `/learn/:categoryId/:topicId`
- **레이아웃**: `max-width: var(--container-max); margin: 0 auto;` (중앙 정렬)
- **사이드바** (280px, sticky):
  - "학습 목차" 헤더
  - 6개 카테고리 목록 (아이콘 + 제목), 활성 카테고리 하이라이트
  - 활성 카테고리의 토픽 목록 (들여쓰기, border-left 활성 표시)
  - 모바일: 토글 버튼으로 열기/닫기
- **콘텐츠 영역**:
  - 브레드크럼 (홈 > 카테고리 > 토픽)
  - 헤더 (아이콘, 제목, 설명)
  - 섹션 탭 (2개 이상일 때 표시)
  - **마크다운 렌더러**: H3, 볼드, 인라인 코드, 코드블록, 인용, 표, 순서/비순서 리스트, 취소선, 단락
  - 하단 네비게이션 (이전/다음 섹션, 다음 토픽/카테고리)

## 기존 페이지

### Home (src/pages/Home.jsx)
6개 섹션:
1. **Hero**: 80vh 다크 그래디언트, CTA 버튼 2개, 통계 4개
2. **학습 콘텐츠**: 6개 카테고리 카드 그리드
3. **도구**: 6개 도구 카드 그리드
4. **학습 특징**: 3개 피처 카드 (체계적 학습, 영어 프레젠테이션, 퀴즈/용어사전)
5. **대상**: 4개 오디언스 카드 (학생, 직장, 교육, 창업)
6. **CTA**: "무료 학습 시작하기" + "과정 신청하기"

### About (src/pages/About.jsx)
- 프로그램 소개, 4개 특징 카드, 18주 타임라인 로드맵, CTA

### CourseDetail (src/pages/courses/CourseDetail.jsx)

- Hero + 2컬럼 (본문 모듈/토픽 + sticky 사이드바) + 이전/다음 과정

### ToolsOverview (src/pages/tools/ToolsOverview.jsx)
- 도구 카드 + 11항목 비교표 + 상황별 6개 추천 카드

### ToolDetail (src/pages/tools/ToolDetail.jsx)
- 동적: `/tools/:toolId`
- Hero + 요금제 카드 + 기능/장점/단점 + 유즈케이스 + 팁 + **가이드 CTA** + 이전/다음 도구
- 가이드 CTA: 그래디언트 카드 (`tool-guide-cta`) → `/tools/:toolId/guide` 링크

### ToolGuide (src/pages/tools/ToolGuide.jsx)
- 동적: `/tools/:toolId/guide`
- LearningPage와 유사한 사이드바+콘텐츠 레이아웃
- **사이드바** (280px, sticky):
  - 도구 소개 페이지 복귀 링크
  - 도구 아이콘 + 이름
  - 5개 섹션 목록 (아이콘+제목), 활성 섹션의 서브섹션 확장
  - 모바일: ☰ 토글 버튼
- **콘텐츠 영역**:
  - 브레드크럼 (홈 > 제작도구 > 도구명 > 종합 가이드)
  - 헤더 (도구 그래디언트 아이콘 + 제목)
  - 섹션 탭 바 (5개 섹션, 아이콘 포함)
  - 서브섹션 탭 바 (2개 이상일 때 표시)
  - 마크다운 렌더링 (`renderMarkdown` 유틸리티)
  - 하단 네비게이션 (이전/다음 서브섹션→섹션 자동 전환)
- **데이터**: `src/data/tool-guides/` → `getGuideByToolId(toolId)`
- **CSS**: `src/styles/tool-guide.css` (~350줄)
