## 사이트 목적
프레젠테이션 발표 방법에 대한 **강의안과 노하우를 전달하는 교육 학습 사이트**.

### 핵심 학습 콘텐츠 (6개 카테고리, 11개 토픽)
1. **프레젠테이션 기초** — 정의, 유형, 기획, 청중 분석
2. **슬라이드 디자인** — CARP 원칙, 레이아웃 패턴, 색상/타이포그래피
3. **스피치와 전달력** — 음성 기법, 보디랭귀지, 긴장 관리
4. **데이터 시각화** — 차트 선택, 인포그래픽, 데이터 스토리텔링
5. **스토리텔링** — 스토리 구조, 감정 연결, TED 분석
6. **영어 프레젠테이션** — 오프닝/전환/클로징 영어 표현

### 보조 학습 기능
- **학습 강의안**: 분야별 PT 강의안 PDF 제공 (기초/디자인/스피치/데이터/스토리텔링), 카드 그리드 + PDF 뷰어 모달
- **퀴즈**: 20문제 (기획, 디자인, 스피치, 도구 등), 정답/오답 피드백 + 해설
- **용어사전**: 36개 프레젠테이션 전문 용어, 검색/카테고리 필터
- **도구 비교**: 6가지 도구 (PowerPoint, Google Slides, Canva, 미리캔버스, Figma, Genially)

### 과정 신청
- 5개 커리큘럼 과정 (기획 → PPT디자인 → 스피치 → 데이터시각화 → 실전발표)
- 과정별 모듈/토픽 상세, 기간, 난이도, 기능 배지

### 부가 서비스
- **제작 의뢰** — 프레젠테이션 제작 대행 의뢰 페이지
- **커뮤니티** — Q&A/자료공유/피드백 게시판, 댓글, 좋아요, 페이지네이션 (Supabase 연동)

## 대상
- 대학생, 취준생: 발표 수업, 팀 프로젝트, 면접
- 직장인, 비즈니스 종사자: 회의, 보고, 제안서 발표
- 교육자, 강사: 교육 자료, 강의 기법
- 창업자, 스타트업: 투자자 피칭, 데모데이

## 주요 특징
- 사이드바 기반 학습 페이지 (OpenClaw 스타일)
- 마크다운 기반 강의 콘텐츠 렌더링
- Google/카카오 OAuth 로그인 (Supabase PKCE)
- 다크모드/라이트모드/오토 테마
- 5가지 컬러 테마 (블루, 틸, 퍼플, 에메랄드, 골드)
- 반응형 디자인 (모바일 ~ 데스크톱)
- SVG + ICO 파비콘
- OG 메타태그 (카카오/소셜 공유 대응)
- JSON-LD 구조화 데이터 (EducationalOrganization)
- AOS 스크롤 애니메이션 (Intersection Observer)
- Code Splitting (React.lazy + Suspense)

## 스크립트

```bash
npm run dev        # 개발 서버 (localhost:5174)
npm run build      # 프로덕션 빌드 → dist/
npm run preview    # 빌드 결과 미리보기
npm run deploy     # 빌드 + gh-pages 배포 (predeploy 자동)
npm run lint       # ESLint 검사
npm run generate-og # OG 이미지 재생성 (sharp)
```
