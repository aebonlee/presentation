# 프로젝트 개요

## 프로젝트명
DreamIT 프레젠테이션 - 효과적인 발표 기법과 시각 자료 제작 학습 사이트

## 도메인
- 프로덕션: https://presentation.dreamitbiz.com
- GitHub Pages 배포

## 기술 스택

| 기술 | 버전 | 용도 |
|------|------|------|
| React | 19.2.0 | UI 프레임워크 |
| Vite | 7.3.1 | 빌드 도구 |
| React Router DOM | 7.13.0 | 클라이언트 사이드 라우팅 |
| Supabase | 2.96.0 | 인증 (Google/카카오 OAuth) |
| Sharp | 0.33.x | OG 이미지 생성 (devDependency) |
| ESLint | 9.39.1 | 코드 린팅 |

## 사이트 목적
- 프레젠테이션 입문 학습 사이트
- PPT, 스피치, 시각디자인, 스토리텔링 교육
- 6가지 프레젠테이션 도구(PowerPoint, Google Slides, Canva, 미리캔버스, Figma, Genially)의 기법, 비용, 활용법 정리
- 5단계 커리큘럼(기획→PPT디자인→스피치→데이터시각화→실전발표)

## 대상
- 대학생, 취준생
- 직장인, 비즈니스 종사자
- 교육자, 강사
- 창업자, 스타트업

## 주요 특징
- 템플릿 제공
- 영상 피드백
- 실전 시뮬레이션
- Google/카카오 OAuth 로그인
- 다크모드/라이트모드/오토 테마
- 5가지 컬러 테마 (블루, 틸, 퍼플, 에메랄드, 골드)
- 반응형 디자인 (모바일~데스크톱)
- OG 메타태그 (카카오 공유 대응)
- AOS 스크롤 애니메이션
- Code Splitting (Lazy Loading)
